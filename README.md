#  AI Flow Application (MERN + React Flow + OpenRouter)

## Project Overview
This is a full-stack MERN application that allows users to:
- Enter a prompt
- Generate an AI response
- Visualize input and output using React Flow
- Save prompt-response data to MongoDB

---

##  Features
-  React Flow-based UI (Input Node → Result Node)
-  AI response generation via OpenRouter API
-  Secure backend (Node + Express)
-  MongoDB integration (Save prompts & responses)
-  Clean and simple user experience

---

##  Tech Stack

### Frontend:
- React.js
- React Flow
- Axios

### Backend:
- Node.js
- Express.js
- MongoDB (Mongoose)

### AI:
- OpenRouter API (Free Model)
